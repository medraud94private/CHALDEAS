# 향후 기능 로드맵

## 상태: 기획 중

## Phase 2: 확장 기능

### 2.1 인물 전용 뷰
- 인물 관계 네트워크 그래프
- 스승-제자 계보 시각화
- 동시대 인물 비교

```
소크라테스 ──┬── 플라톤 ──── 아리스토텔레스
             │
             └── 크세노폰
```

### 2.2 날짜+장소 복합 검색
- 달력 UI로 날짜 선택
- 미니 지구본으로 장소 선택
- "490 BCE 그리스 일대" 같은 범위 검색

### 2.3 다국어 지원
- Google Translate API 연동
- 사용자 언어 설정 저장
- 번역 캐싱

---

## Phase 3: 고급 기능

### 3.1 이벤트 연결선 시각화
- 인과관계 화살표
- 동시대 점선 연결
- 영향력 흐름 애니메이션

```
마라톤 전투 ──→ 살라미스 해전 ──→ 플라타이아 전투
    │               │
    └───────────────┴──→ 그리스 황금기
```

### 3.2 AI 연관 추천
LOGOS를 활용한 개인화 추천:
- "이 사건에 관심이 있다면..."
- "비슷한 시대의 다른 지역..."
- 학습 경로 추천

### 3.3 사용자 북마크
- 로컬 스토리지 저장
- 개인 컬렉션 생성
- 학습 기록 트래킹

## Phase 4: 커뮤니티 기능

### 4.1 기여자 시스템
