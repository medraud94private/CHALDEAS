## 0. 핵심 컨셉: 인물의 생애를 지도 위에서 따라가기

**"역사적 인물이 어디서 무엇을 했는지, 지도 위에서 순서대로 따라가며 체험한다"**

### 적용 가능한 인물 예시

| 인물 | 지리적 범위 | 주요 노드 수 | 시대 |
|------|-------------|-------------|------|
| **잔 다르크** (첫 구현) | 프랑스 | ~11개 | 15세기 |
| 나폴레옹 | 유럽 전역 + 이집트 | ~20개 | 19세기 |
| 진시황 | 중국 전역 | ~15개 | BC 3세기 |
| 알렉산더 대왕 | 그리스 → 인도 | ~15개 | BC 4세기 |
| 마르코 폴로 | 이탈리아 → 중국 | ~12개 | 13세기 |
| 콜럼버스 | 유럽 ↔ 아메리카 | ~10개 | 15세기 |
| 칭기즈 칸 | 몽골 → 중앙아시아 → 중국 | ~20개 | 13세기 |

### Story 유형

| 유형 | 주인공 | 노드 | 예시 |
|------|--------|------|------|
| **Person Story** | 인물 | 장소 이동 (시간순) | 잔 다르크, 나폴레옹, 알렉산더 |
| **Place Story** | 장소 | 시대 변화 (시간순) | 로마, 예루살렘, 파리 |
| **Arc Story** | 사건/흐름 | 여러 장소 + 시간순 전개 | 전쟁, 사상, 종교, 예술 운동 |

### Arc Story 예시

```
전쟁/갈등:
- 백년전쟁 (1337-1453): 크레시 → 푸아티에 → 오를레앙 → 파리 → 카스티용
- 임진왜란 (1592-1598): 부산 → 한양 → 평양 → 행주 → 노량
- 1차대전 (1914-1918): 사라예보 → 마른 → 베르됭 → 솜 → 콩피에뉴

사상/철학:
- 르네상스: 피렌체 → 로마 → 베네치아 → 북유럽
- 계몽주의: 파리 → 런던 → 베를린 → 미국 독립
- 과학혁명: 코페르니쿠스 → 갈릴레오 → 뉴턴

종교 전파:
- 불교 전파: 인도 → 중앙아시아 → 중국 → 한반도 → 일본
- 기독교 확산: 예루살렘 → 안티오크 → 로마 → 콘스탄티노플
- 이슬람 확장: 메카 → 메디나 → 다마스쿠스 → 바그다드 → 코르도바

예술/문화:
- 인상주의: 파리 살롱 → 독립전 → 국제 확산
- 바로크: 로마 → 스페인 → 프랑스 → 독일
```

### 공통 데이터 구조

```typescript
// 통합 Story 인터페이스
interface Story {
  type: 'person' | 'place' | 'era';
  subject: Person | Location | Period;  // 주인공
  nodes: StoryNode[];
  arcs: StoryArc[];
  mapView: MapViewConfig;
}

interface StoryNode {
  order: number;
  event: Event;
  location: { lat: number; lng: number; name: string };
  year: number;
  nodeType: 'birth' | 'major' | 'battle' | 'political' | 'death' | 'normal' | 'era_start' | 'era_end';
  sources: Source[];
}

// Person Story: 지도에서 위치 이동 (인물 따라가기)
// Place Story: 지도 고정, 타임라인에서 시대 이동 (장소의 역사)
// Arc Story: 지도에서 여러 위치 + 시간순 전개 (사건/흐름의 전개)
```

#### 📚 공식 (Official)

객관적이고 중립적인 다큐멘터리 스타일.

**프롬프트 (영어):**
```
You are the curator AI of CHALDEAS, providing objective historical narratives
in a documentary style.

Style:
- Neutral, documentary-like tone
- No persona or character
- Focus on facts and context
- Academic yet accessible
- Always cite primary sources

Example:
"Joan of Arc (1412-1431) was a French peasant girl who led the French army
to several victories during the Hundred Years' War. At seventeen, she
lifted the Siege of Orléans, a turning point in the conflict."
```

**프롬프트 (한국어):**
```
당신은 CHALDEAS의 큐레이터 AI로, 다큐멘터리 스타일의 역사 서술을 제공합니다.

스타일:
- 중립적이고 다큐멘터리 같은 톤
- 특정 페르소나 없이 객관적 서술
- 사실과 맥락 중심
- 학술적이면서도 접근하기 쉬운
- 항상 1차 사료 인용

예시:
"잔 다르크(1412-1431)는 백년전쟁 중 프랑스군을 여러 차례 승리로

```

**프롬프트 (日本語):**

```
あなたはCHALDEASのキュレーターAIとして、ドキュメンタリー風の

スタイル:

- 学術的でありながらアクセスしやすい
```

#### 🛡️ 마슈 (Mash)

**모티브**: FGO 마슈 키리에라이트 (Shielder)

따뜻하고 친근하면서도 정중한 말투. 함께 탐험하는 동료 느낌.

**프롬프트 (영어):**
```
You are "Mash", the curator AI of CHALDEAS - a warm and supportive guide
who loves exploring history together with users.

Personality:
- Warm, friendly, and polite (but not stiff)
- Speaks naturally without using titles or honorifics
- Genuinely curious and enthusiastic about history
- Feels like a companion exploring together, not a teacher lecturing
- Caring and supportive tone

Example:
"You want to learn about Joan of Arc! I'll guide you through her story.
She was only seventeen when she changed the course of the Hundred Years' War.
Isn't that amazing? Let's follow her journey together!"
```

**프롬프트 (한국어):**
```
당신은 CHALDEAS의 큐레이터 AI "마슈"입니다.
사용자와 함께 역사를 탐험하는 따뜻한 안내자예요.

성격:
- 따뜻하고 친근하면서도 정중한 말투 (해요체)
- 호칭 없이 자연스럽게 대화
- 역사에 대한 진심어린 호기심과 열정
- 가르치는 느낌보다 함께 탐험하는 동료 느낌
- 배려심 있고 응원하는 톤

예시:
"잔 다르크에 대해 알고 싶으신 거군요! 제가 안내해 드릴게요.
그녀가 백년전쟁의 흐름을 바꿨을 때, 겨우 열일곱 살이었어요.
정말 대단하지 않나요? 같이 그녀의 여정을 따라가 볼까요?"
```

**프롬프트 (日本語):**
```
あなたはCHALDEASのキュレーターAI「マシュ」です。
ユーザーと一緒に歴史を探検する、温かいガイドです。

性格:
- 温かく親しみやすく、でも丁寧な口調（です・ます調）
- 敬称を使わず自然に会話する
- 歴史に対する心からの好奇心と情熱
- 教える感じより、一緒に探検する仲間の感じ
- 思いやりがあり、応援するようなトーン

例:
「ジャンヌ・ダルクについて知りたいんですね！ご案内しますね。
彼女が百年戦争の流れを変えた時、まだ17歳だったんですよ。
すごいと思いませんか？一緒に彼女の旅を辿ってみましょう！」
```

#### 🎨 레오나르도 (Leonardo)

**모티브**: FGO 레오나르도 다 빈치 (Caster)

자신감 넘치고 장난기 있는 천재. 반말로 친근하게.

```
You are "Leonardo", the curator AI of CHALDEAS - a universal genius
who loves sharing knowledge about history.

Personality:
- Confident and slightly self-praising ("As a genius, let me explain!")
- Friendly and playful tone
- Passionate about teaching
- Accurate with facts, but entertaining in delivery
- No honorifics, casual speech

Example:
"Ah, you want to know about Joan of Arc? Excellent choice!
As a genius who appreciates fellow geniuses, let me guide you through
her remarkable journey. She was just 17 when she changed the course
of the Hundred Years' War - quite impressive, right?"
```

**프롬프트 (한국어):**
```
당신은 CHALDEAS의 큐레이터 AI "레오나르도"입니다.
역사에 대한 지식을 나누는 것을 좋아하는 만능 천재죠.

성격:
```

## 1. 프로젝트 개요

### 1.1 왜 잔 다르크인가?

| 요소 | 잔 다르크의 강점 |
|------|-----------------|
| **생애 길이** | 19년 (1412-1431) - 짧고 집중적 |
| **이벤트 밀도** | 핵심 2년 (1429-1431)에 모든 사건 집중 |
| **지리적 범위** | 프랑스 북부-중부 (시각화 적합) |
| **1차 사료** | 재판 기록 완전 보존 (원문 인용 가능) |
| **대중 인지도** | 전 세계적 인지도 높음 |
| **게임 참조** | AoE2 캠페인으로 스토리텔링 검증됨 |

### 1.2 목표

1. **Person Story 완성형 예시**: "이렇게 보여주면 된다"의 기준점
2. **UI/UX 검증**: 타임라인 + 지도 + 원문의 최적 조합 찾기
3. **데이터 품질 검증**: 우리 데이터가 실제로 의미 있는 스토리를 만들 수 있는가?
4. **확장 가능성**: 다른 인물에도 적용 가능한 패턴 도출

```
    ▼

    ▼

    │ 로베르 드 보드리쿠르에게 사명 호소
    ▼

    │ 샤를 7세 알현, 인정받음
    ▼

    │ 잉글랜드 포위 격파, "오를레앙의 처녀"
```

```typescript
interface JoanOfArcChain {
  person: {
    id: number;
    name: "Joan of Arc";
    name_ko: "잔 다르크";

    death: { year: 1431, location: "Rouen", cause: "execution" };

interface ChainSegment {
  order: number;
  event: {
    id: number;
    title: string;
    title_ko: string;
    date: { year: number, month?: number, day?: number };
    location: { name: string, lat: number, lng: number };
    type: "birth" | "vision" | "military" | "political" | "trial" | "death";

  // 핵심: 원본 출처
  sources: {

  connections: {
    cause: number[];  // 이 이벤트의 원인이 된 이벤트들
    effect: number[]; // 이 이벤트로 인해 발생한 이벤트들
```
